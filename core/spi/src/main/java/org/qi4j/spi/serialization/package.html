Serialization SPI.

This package contains specialized serialization APIs for the {@literal JSON} and {@literal XML} formats. See {@link org.qi4j.spi.serialization.JsonSerialization}, based on {@literal javax.json}, and {@link org.qi4j.spi.serialization.XmlSerialization}, based on {@literal javax.xml}.

This package also contains base implementations, mixins and helpers for serialization API implementations:

Base implementations

  • Use {@link org.qi4j.spi.serialization.AbstractTextSerializer} and {@link org.qi4j.spi.serialization.AbstractTextDeserializer} as a basis to implement the serialization API for text representations.
  • Use {@link org.qi4j.spi.serialization.AbstractBinarySerializer} and {@link org.qi4j.spi.serialization.AbstractBinaryDeserializer} as a basis to implement the serialization API for binary representations.
  • Use {@link org.qi4j.spi.serialization.AbstractSerializer} and {@link org.qi4j.spi.serialization.AbstractDeserializer} if you need to handle text/binary conversion yourself.

Mixins

  • {@link org.qi4j.spi.serialization.BuiltInConverters} provides built-in {@link org.qi4j.api.serialization.Converter}s for types supported by the Qi4j Runtime.

Helpers

  • {@link org.qi4j.spi.serialization.SerializationSettings} can be used to register custom Converters at assembly time. Serialization implementations might extend this with additional specialized settings, see their respective documentation for details.
