LevelDB EntityStore.

LevelDB is a fast key-value storage library written at Google that provides an ordered mapping from string keys to string values.

By default use the native implementation through JNI bindings and fallback to the pure java implementation if not available on the current platform. Used implementation can be forced using the {@link LevelDBEntityStoreConfiguration#flavour()} property.

See:

  • LevelDB https://code.google.com/p/leveldb/
  • JNI bindings https://github.com/fusesource/leveldbjni
  • Pure Java implementation https://github.com/dain/leveldb
